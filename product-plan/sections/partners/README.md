# Partners

## Overview

The Partners module manages external business introducers who onboard subscribers on the platform.

## Components Provided

- `PartnersDashboard` — Main view
- `PartnerList` — Partner list with filters
- `AddPartner` — 4-step onboarding
- `EditPartner` — Edit modal
- `PartnerDetail` — Detail page with 5 tabs

## Callback Props

| Callback | Description |
|----------|-------------|
| `onCreate` | Create new partner |
| `onView` | View partner details |
| `onToggleStatus` | Toggle active/inactive |
| `onViewIncidents` | View subscriber incidents |
