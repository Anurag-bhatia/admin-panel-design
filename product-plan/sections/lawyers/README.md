# Lawyers

## Overview

The Lawyers module manages the platform's external legal network with onboarding, credentialing, and compliance tracking.

## Components Provided

- `Lawyers` — Main management view
- `LawyerList` — Lawyer list with tabs
- `LawyerTable` — Lawyer data table
- `LawyerRow` — Individual lawyer row
- `LawyerForm` — 7-step onboarding wizard
- `LawyerProfile` — Profile with 4 tabs

## Callback Props

| Callback | Description |
|----------|-------------|
| `onView` | View lawyer profile |
| `onEdit` | Edit lawyer details |
| `onDeactivate` | Deactivate lawyer |
| `onReactivate` | Reactivate lawyer |
| `onAdd` | Add new lawyer |
| `onViewDocument` | View KYC document |
| `onRaiseInvoice` | Raise invoice |
