# Reports

## Overview

The Reports module provides comprehensive analytics dashboards across all business domains.

## Components Provided

- `ReportsDashboard` — Main container
- `ExecutiveDashboard` — Summary view
- `FilterBar` — Global filters
- `MetricCard` — Reusable metric card
- Domain-specific tab components

## Callback Props

| Callback | Description |
|----------|-------------|
| `onTabChange` | Switch report tabs |
| `onFilterChange` | Apply filters |
| `onMetricClick` | Drill down from metric |
| `onExport` | Export report |
