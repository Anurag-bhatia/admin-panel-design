# Team

## Overview

The Team module manages employees, teams, permissions, and access control.

## Components Provided

- `TeamManagement` — Main view
- `EmployeeTable` — Employee list
- `EmployeeOnboardingWizard` — 2-step onboarding
- `EditEmployeeModal` — Edit modal
- `DeactivateConfirmDialog` — Deactivation confirmation
- `PermissionsPage` — Permissions configuration
- `TeamsTable` — Teams list
- `CreateTeamModal` — Create team modal
- `EditTeamModal` — Edit team modal
- `EmployeeDetailView` — Employee detail
- `TeamDetailView` — Team detail

## Callback Props

| Callback | Description |
|----------|-------------|
| `onAddEmployee` | Add new employee |
| `onEditEmployee` | Edit employee |
| `onDeactivateEmployee` | Deactivate employee |
| `onManagePermissions` | Open permissions |
| `onSavePermissions` | Save permissions |
| `onCreateTeam` | Create team |
| `onEditTeam` | Edit team |
| `onToggleTeamStatus` | Toggle team status |
