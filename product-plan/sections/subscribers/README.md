# Subscribers

## Overview

The Subscribers module manages all customers actively or historically subscribed to the platform's services. It connects sales, operations, and finance into a single traceable view.

## Components Provided

- `SubscribersDashboard` — Main list view
- `SubscriberList` — Subscriber table with filters
- `AddSubscriberModal` — Add subscriber form
- `BulkUploadModal` — Bulk Excel upload
- `SubscriberDetail` — Full detail page with 7 tabs

## Callback Props

| Callback | Description |
|----------|-------------|
| `onViewDetails` | View subscriber details |
| `onEdit` | Edit subscriber |
| `onAddSubscriber` | Add new subscriber |
| `onBulkUpload` | Bulk upload |
| `onManageSubscription` | Manage subscription |
| `onUploadDocument` | Upload document |
